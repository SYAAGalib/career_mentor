# CareerMentor AI Project Proposal

## Project Goal

This platform helps people find career paths using commitment. It stops the habit of switching interests before mastering them.

## Operation

Users begin with a path choice after a personality matching session. This choice stays locked for sixty days. A person must finish eighty percent of a roadmap to change their direction.

## Structure

The roadmap looks like a tree. It has sections and branches. A user must complete one branch to see the next level. This builds discipline and focus.

## Features

Matching happens through personality traits and values. AI mentors provide feedback as a coach or guide. Simulated work tasks show what a job requires.

Specific spotlights feature professionals from Bangladesh. A tool builds resumes from finished work. AI checks user mood and changes the pace of lessons.

## Implementation

Roadmaps are dynamic. AI builds a path for each person based on goals. Quizzes check mastery before a user can skip any topic. The system uses modular blocks to grow.

## Benefit

The platform builds consistency. It removes wasted effort on known topics. It turns learning into assets.
